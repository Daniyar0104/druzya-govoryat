Настройка Supabase

Сейчас приложение работает в demo-режиме — все данные хранятся в памяти браузера и сбрасываются при обновлении страницы.

Чтобы данные сохранялись постоянно, введите ключи от вашего бесплатного проекта Supabase.

Как получить ключи — пошаговая инструкция

  1. Зайдите на supabase.com → "Start your project" → регистрация бесплатна
  2. Нажмите "New Project", введите название druzya-govoryat и придумайте пароль
  3. Дождитесь создания проекта (~1 минута)
  4. Перейдите в Settings → API
  5. Скопируйте Project URL и anon public key в поля выше
  6. Перейдите в SQL Editor и выполните файл schema.sql из скачанного архива
