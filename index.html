Что осталось сделать для полной автоматизации
Сейчас приложение уже работает как единый вход сотрудников + CRM база гостей по бонусной карте. Но автоматическая подтяжка гостя из внешней бонусной системы возможна только если у Prime Hill есть API, webhook, CSV-выгрузка или интеграционный доступ.
  • Если у Prime Hill есть API — можно делать автоматический запрос по номеру карты.
  • Если API нет, можно настроить ручной импорт CSV списка гостей.
  • Если есть Telegram-чат броней или РобоХост API — следующий этап можно связать бронь, стол и бонусную карту.
